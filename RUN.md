# 🚀 Gym Management System - Hızlı Başlangıç

## Backend Çalıştırma
```bash
cd backend
./mvnw.cmd spring-boot:run
```
Backend: http://localhost:8082

## Frontend Çalıştırma
```bash
cd frontend
npm start
```
Frontend: http://localhost:3000
